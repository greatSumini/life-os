# fitness-review Skill

운동 기록을 종합 분석하여 진행 상황을 리포트하는 스킬이다.

## 파일 경로

- 운동 사전: `fitness/exercises.toml`
- 운동 로그 디렉터리: `fitness/logs/`
- 인바디 기록 디렉터리: `fitness/inbody/`
- 트레이너 리포트 디렉터리: `fitness/reports/`
- 사용자 프로필: `fitness/profile.toml`
- 분석 기준: `fitness/CLAUDE.md`

## 실행 절차

### Step 0. 현재 한국 날짜 확인 및 기간 설정

```bash
TZ=Asia/Seoul date '+%Y-%m-%d'
```

사용자 입력에서 기간을 파악한다:
- `1w` (기본): 최근 7일
- `2w`: 최근 14일
- `1m`: 최근 30일

### Step 1. 데이터 수집

다음을 병렬로 읽는다:

**1-1. 해당 기간의 운동 로그**

```bash
ls fitness/logs/ | sort -r
```

기간에 해당하는 파일들을 Read 툴로 읽는다.

**1-2. 운동 사전**

`fitness/exercises.toml`을 Read 툴로 읽는다.

**1-3. 해당 기간의 인바디 기록**

```bash
ls fitness/inbody/ | sort -r
```

기간 내 인바디 파일이 있으면 Read 툴로 읽는다. 기간 직전 기록도 비교를 위해 읽는다.

**1-4. 사용자 프로필**

`fitness/profile.toml`을 Read 툴로 읽어 목표와 취약점을 확인한다.

**1-5. 해당 기간의 트레이너 리포트**

기간 내 리포트가 있으면 Read 툴로 읽는다.

### Step 2. 분석

다음 항목을 분석한다:

**2-1. 근육 그룹별 볼륨 분석**

exercises.toml과 조인하여 근육 그룹별 총 세트 수를 계산한다.
MEV/MAV 기준 대비 충족 여부를 판단한다.

**2-2. 점진적 과부하 추세**

같은 운동에 대해 기간 내 weight_kg 변화를 추적한다.
- 증가 추세: 정상적 과부하
- 정체: 플래토 경고
- 감소: 회복 필요 또는 폼 교정 중

**2-3. 운동 빈도 분석**

- 총 운동 일수 / 기간 일수
- PT vs 개인 운동 비율
- 휴식일 패턴

**2-4. 인바디 변화 (데이터 있는 경우)**

- 골격근량 변화
- 체지방률 변화
- 부위별 근육량 변화 (특히 목표 근육 관련 부위)

**2-5. 목표 달성도 평가**

profile.toml의 목표(넓은 등판)와 취약점을 기준으로:
- 목표 근육(등) 볼륨이 충분했는지
- 취약점 근육에 충분한 볼륨을 할당했는지
- 인바디에서 관련 부위 성장이 보이는지

### Step 3. 리포트 출력

```
## 피트니스 리포트 (2026-04-10 ~ 2026-04-16, 1주)

### 운동 요약
- 운동 일수: 5/7일 (PT 3회, 개인 2회)
- 휴식일: 월, 일

### 근육 그룹별 주간 볼륨
| 그룹 | 세트 수 | MEV(10) | MAV(20) | 상태 |
|------|---------|---------|---------|------|
| 등 | 22 | ✓ | ✓ | 목표 충족 |
| 가슴 | 12 | ✓ | - | 유지 |
| 어깨 | 8 | ✗ | - | MEV 미달 |
| 팔 | 6 | ✗ | - | MEV 미달 |
| 하체 | 14 | ✓ | - | 유지 |
| 코어 | 3 | ✗ | - | 취약점인데 부족 ⚠️ |

### 점진적 과부하 추적
- 랫풀다운: 47.5kg → 50kg (+2.5kg) ✓
- 시티드 로우: 40kg → 40kg (정체 — 다음 주 42.5kg 시도 권장)
- 스쿼트: 80kg → 82.5kg (+2.5kg) ✓

### 인바디 변화 (해당 기간 측정 있는 경우)
- 골격근량: 33.5kg → 34.1kg (+0.6kg)
- 체지방률: 20.3% → 19.7% (-0.6%p)
- 몸통 근육량: 25.0kg → 25.8kg (+0.8kg) — 등 발달 긍정적

### 목표(넓은 등판) 진행도
- ✅ 등 볼륨 MAV 충족
- ⚠️ 후면삼각근(취약점) 볼륨 부족 — 페이스풀/리버스 플라이 추가 필요
- ✅ 인바디 몸통 근육량 증가 확인

### 다음 주 방향
1. 어깨(특히 후면삼각근) 볼륨 +6세트 이상 확보
2. 코어 운동 주 2회 이상 포함
3. 시티드 로우 무게 증량 시도 (42.5kg)
```
