---
name: 스킬 네이밍 컨벤션
description: 같은 도메인의 스킬들은 공통 프리픽스로 그룹화
type: feedback
---

같은 도메인에 스킬이 여러 개일 때 공통 프리픽스를 붙여 분류가 쉽도록 한다. (예: fitness-log-workout, fitness-log-inbody, fitness-recommend)

**Why:** 사용자가 `/fitness-`만 타이핑하면 관련 스킬이 모두 자동완성되도록 하기 위함.

**How to apply:** 새 도메인의 스킬을 3개 이상 만들 때 프리픽스 네이밍을 적용할 것.
