---
name: 데이터 정규화 선호
description: 데이터 구조 설계 시 중복을 최소화하는 정규화를 선호함
type: feedback
---

데이터 구조 설계 시 같은 정보가 여러 곳에 반복 정의되는 것을 지양한다. 참조 데이터는 한 곳에서 정의하고 id로 참조하는 방식을 선호.

**Why:** 운동 기록 설계 시 매 로그마다 타겟 근육을 반복하는 대신, 운동 사전(exercises.toml)을 별도로 두고 id로 참조하는 구조를 직접 제안함.

**How to apply:** 새로운 데이터 구조를 설계할 때, 반복될 수 있는 정보는 별도 참조 파일로 분리하고 id 기반 참조를 사용할 것.
